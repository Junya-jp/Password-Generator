# Password-Generator
## Description
This project is a password generator that acts as a way to help obtain secure passwords. These passwords are reandomized given a  set of criteria.


## Installation

All you need to run this project is a web browser. It acts as an application within the website.
## Usage
When user clicks on generate password button they will be met with a set of prompts. The first action is choosing the length for security the minimum length is 8 and maximum length is 128. Choosing otherwise will give user a message to try again. Then the user will set the criteria of their password based on prompts choosing whether to include Upper or lower case or both, special characters, and numbers. Choosing all criteria will create a more secure password but user is permitted to choose any combination they want.
